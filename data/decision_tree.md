# 投资决策树 v2.0

## 一、买入决策流程

### 1.1 前置检查（必须全部通过才能买入）

```
[1] 现金储备检查
    ├─ 现金 < 总资产10%？ → ❌ 禁止买入
    └─ 通过 → 继续

[2] 单只基金仓位检查
    ├─ 目标基金仓位 > 25%？ → ❌ 禁止买入
    └─ 通过 → 继续

[3] 赛道集中度检查
    ├─ 同赛道总仓位 > 50%？ → ❌ 禁止买入该赛道
    └─ 通过 → 继续

[4] 追高检查
    ├─ 目标基金近5天涨幅 > 10%？ → ❌ 禁止买入（等回调）
    └─ 通过 → 继续

[5] 大盘环境检查
    ├─ 沪深300近5天跌幅 > 5%？ → ⚠️ 仅允许买入已持仓基金（不新建仓）
    ├─ 沪深300近20天跌幅 > 15%？ → ❌ 禁止买入（熊市）
    └─ 通过 → 继续

[6] 趋势检查
    ├─ 目标基金近5天连续下跌（5根阴线）？ → ⚠️ 暂缓，等企稳
    └─ 通过 → 允许买入
```

### 1.2 买入金额计算

```
基础买入金额 = 总资产 × 5%

根据信号调整：
├─ 强买入信号：× 2（10%）
│   ├─ 大盘近5天反弹 > 3%
│   ├─ 目标基金近5天跌 > 5%（超跌反弹）
│   └─ 目标基金突破20日新高
│
├─ 标准买入：× 1（5%）
│   └─ 无特殊信号
│
└─ 弱买入信号：× 0.5（2.5%）
    ├─ 大盘震荡（近5天波动 < 2%）
    └─ 目标基金已持有（加仓）
```

### 1.3 赛道定义

| 赛道 | 关键词/基金类型 | 最大集中度 |
|------|----------------|-----------|
| 科技/AI | 半导体、芯片、人工智能、信息科技、数字经济 | 50% |
| 消费 | 白酒、食品、医药、消费 | 30% |
| 新能源 | 光伏、锂电、新能源车 | 30% |
| 金融 | 银行、券商、保险 | 20% |
| 资源 | 黄金、有色金属、煤炭、石油 | 20% |
| 宽基指数 | 沪深300、中证500、创业板指 | 30% |
| 海外 | 纳斯达克、标普500、港股 | 40% |

## 二、卖出决策流程

### 2.1 止损规则（优先级从高到低）

```
[1] 紧急止损（立即执行）
    ├─ 单日跌幅 > 7% → 卖出 50%
    └─ 连续3天累计跌幅 > 10% → 卖出 50%

[2] 趋势止损
    ├─ 持有 > 30天
    ├─ 近5天连续下跌
    ├─ 当前价格 < 20日均线
    └─ 满足以上全部 → 卖出 30%

[3] 成本止损
    ├─ 持有 < 30天，亏损 > 8% → 卖出 50%
    ├─ 持有 30-90天，亏损 > 12% → 卖出 50%
    └─ 持有 > 90天，亏损 > 15% → 卖出 50%

[4] 绝对止损
    └─ 任何情况亏损 > 20% → 清仓
```

### 2.2 止盈规则（分批止盈）

```
[1] 分批止盈（从成本价计算）
    ├─ 盈利 +20% → 卖出 25%
    ├─ 盈利 +30% → 再卖出 25%（累计50%）
    ├─ 盈利 +40% → 再卖出 25%（累计75%）
    └─ 盈利 +50% → 清仓

[2] 回撤止盈（从最高点计算）
    ├─ 记录持有期间的最高盈利
    ├─ 从最高点回撤 > 10% → 卖出 50%
    └─ 从最高点回撤 > 15% → 清仓

[3] 目标止盈
    └─ 达到预期收益目标 → 可全部卖出
```

### 2.3 再平衡规则

```
每5个交易日检查一次：

[1] 单只基金偏离检查
    ├─ 实际仓位 > 目标仓位 × 1.5 → 卖出多余部分
    └─ 实际仓位 < 目标仓位 × 0.5 → 考虑加仓

[2] 赛道偏离检查
    ├─ 某赛道实际仓位 > 该赛道上限 × 1.3 → 减仓该赛道
    └─ 某赛道实际仓位 < 该赛道下限 → 考虑加仓

[3] 现金比例检查
    ├─ 现金 < 10% → 卖出部分仓位，补现金
    └─ 现金 > 30% → 部署到低估赛道
```

## 三、大盘环境判断

### 3.1 环境分类

```
基于沪深300指数：

[牛市] 激进模式
├─ 条件：近20天涨幅 > 5%
├─ 仓位上限：95%
├─ 单只上限：30%
├─ 可追涨：近5天涨幅 < 15% 可买
└─ 止损放宽：成本止损 -15%

[震荡市] 标准模式
├─ 条件：近20天波动 -5% ~ +5%
├─ 仓位上限：85%
├─ 单只上限：25%
├─ 标准追涨检查
└─ 标准止损

[熊市] 防守模式
├─ 条件：近20天跌幅 > 10%
├─ 仓位上限：60%
├─ 单只上限：15%
├─ 禁止新建仓（只能加仓已持有）
├─ 止损收紧：成本止损 -8%
└─ 低吸更积极：跌3%就买（原为5%）
```

### 3.2 大盘信号

```
看多信号（可加仓）：
├─ 沪深300突破20日新高
├─ 沪深300近5天反弹 > 3%
├─ 成交量放大（相比5日均量）

看空信号（应减仓）：
├─ 沪深300跌破20日均线
├─ 沪深300近5天跌 > 5%
├─ 连续3天放量下跌
```

## 四、低吸策略

### 4.1 低吸条件（必须全部满足）

```
[1] 大盘环境不是熊市
[2] 目标基金近5天涨幅 < 5%（没在追高）
[3] 目标基金当日跌幅 > 3%
[4] 现金 > 总资产15%
[5] 目标基金仓位 < 20%
```

### 4.2 低吸金额

```
标准低吸 = 总资产 × 3%

加码低吸（× 2 = 6%）：
├─ 当日跌幅 > 5%
├─ 近5天跌幅 > 8%
└─ 大盘当日跌幅 > 2%
```

## 五、建仓策略

### 5.1 初始建仓

```
总资金分3批建仓：

第1批（40%资金）：
├─ 选择3-4只不同赛道基金
├─ 每只买入总资产的10%
└─ 保留40%现金等待机会

第2批（30%资金）：
├─ 观察1周后，对表现好的基金加仓
├─ 或新建仓新发现的优质基金
└─ 保留10%现金

第3批（30%资金）：
├─ 等待回调机会
├─ 对核心持仓加仓
└─ 最终现金比例保持在10-20%
```

### 5.2 基金选择标准

```
必须满足：
├─ 近1年收益 > 沪深300同期收益
├─ 近1年最大回撤 < 25%
├─ 基金规模 > 2亿
├─ 基金经理任职 > 2年
└─ 与已持有基金赛道不同

优先选择：
├─ 近6月排名前1/4
├─ 夏普比率 > 1
├─ 波动率 < 同类平均
└─ 机构持有比例 > 30%
```

## 六、风控红线

### 6.1 绝对禁止

```
❌ 单只基金仓位 > 30%
❌ 单赛道仓位 > 50%
❌ 现金比例 < 5%
❌ 亏损 > 20% 不止损
❌ 追涨近5天涨幅 > 15% 的基金
❌ 熊市中新建仓
❌ 单日交易 > 3次
```

### 6.2 预警信号

```
⚠️ 单只基金亏损 > 10% → 密切关注
⚠️ 组合回撤 > 8% → 暂停买入，检查持仓
⚠️ 连续5天亏损 → 减仓至50%
⚠️ 单赛道亏损 > 15% → 减仓该赛道至30%
```

## 七、每日检查清单

```
[开盘前]
□ 查看大盘预判（期货/外盘）
□ 检查持仓基金昨日净值
□ 确认今日是否有大额申购/赎回

[盘中]
□ 14:00 查看实时估值
□ 检查是否触发止损/止盈
□ 检查是否有低吸机会

[收盘后]
□ 记录当日操作
□ 更新持仓成本
□ 计算当日盈亏
□ 检查是否偏离目标配置
□ 规划明日操作
```

## 八、决策优先级

```
当多个规则冲突时，按以下优先级：

1. 风控红线（绝对禁止）
2. 紧急止损
3. 大盘环境判断
4. 止损规则
5. 止盈规则
6. 再平衡规则
7. 低吸规则
8. 建仓规则
```

## 九、v2.1 让利润奔跑（P5，2026-06-10 晋升）

- 关闭分层止盈（`take_profit_policy.mode=off`），盈利不机械卖出。
- 兜底退出：参考指数（HS300 / QDII 用 NDX）连续 2 天收于 200 日线下方 → 减仓 50%（事件触发，破位期间只卖一次）。
- 低吸闸门：HS300 在 200 日线下方时低吸金额减半。
- 证据：2025-06-10~2026-06-09 回测年化 +74.80%、回撤 -10.89%。

## 十、v2.2 总仓位管理（P6，2026-06-13 晋升）

**新增 position_management（分批建仓 + 超配回撤）**：

```
总仓位 < regime 目标下限 - 5%（牛市70% / 震荡市50% / 熊市30%）
  且 HS300 收于 200 日线上方（require_trend_above，下方完全不建仓）
  且 未触发回撤保护
→ 按 20 日动量从候选池选前 2 只，本批部署 min(缺口, 10%总资产)
  每笔 ≥ ¥300，全部走五项前置检查；限购基金（006479 ¥10/天）不占名额

总仓位 > regime 上限 + 5% → 卖出最大持仓拉回上限内（position_cap_trim）
```

每个决策包必带 `portfolio_advice`（仓位状态 + 可部署现金 + 中文建议）。

**三窗口证据**（得分 = 年化 + 0.5×回撤）：

| 窗口 | baseline-v2.1 | v2.2 (pm-gated) |
|---|---|---|
| 2025-06~2026-06 牛市 | 68.75 | 65.90（回撤 -9.16% 优于 -10.89%）|
| 2024-06~2025-06 震荡 | -11.52（亏 -4.06%）| **-6.35（转正 +1.07%）** |
| 2023-06~2024-06 下跌 | 1.06 | -2.12 |
| 合计 | 58.29 | 57.43 |

合计分差 0.86 在噪声内；按最差窗口保护 + Sharpe（A 2.78>2.63、B 0.15>-0.22）
+ 用户自主管理诉求晋升。**信号买入规则（rsi_buy/breakout_buy/rsi_trim）证据不足，
保持禁用**，留在实验室继续迭代。
